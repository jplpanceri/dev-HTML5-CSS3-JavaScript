Formulário

Tag form é utilizada para criar formulários onde se pode colocar todo tipo de tag dentro.
Atributo action é utilizado para dizer o endereço para onde os dados do formulário serão enviados.
Atributo method é utilizado para definir a forma de envio dos dados, pode ter o valor post(mostra somente o valor de destino na barra de endereço) e get(mostra todos os dados).
Tag input é utilizada para capiturar dados e ela possui os atributos type, name e value.
Tag select é utilizada para criar uma caixinha com várias opções e que utiliza o atributo name.
Tag option é utilizada para dar as opções da caixinha do select e que utiliza o atributo value.
Atributo type define o tipo de entrada e tem os valores text(texto), password(texto oculto), radio(círculo de marcar, se a resposta for única, o valor do atributo name deve ser o mesmo pra todos), checkbox(quadrado de marcação, pro caso de existir limitação deve-se definir na página de destino), button(botão para clique) e submit(botão de envio).
Atributo name define o nome que a entrada vai receber quando ela for para a outra página.
Atributo value tem 2 funções a primeira é dar o valor a ser recebido pelo name quando utilizamos os tipos de entrada radio e checkbox, também dando o valor para a tag option. A outra é colocar o nome dentro dos tipos de entrada button e submit.
Tag textarea é uma caixa de entrada de texto que pode ser expandida manualmente ou a medida que vai recebendo dados.
