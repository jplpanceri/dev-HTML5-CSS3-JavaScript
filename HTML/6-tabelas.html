A tag table é utilizada para fazer tabelas.
O atributo boarder serve para dar o valor da borda da tabela em pixel px.
A tag tr table row cria uma linha para a tabela.
A tag tb table data cria uma célula para a tabela.
A tag th table header cria uma célula com o texto em negrito e centralizado.
O artibuto colspan mescla células na horizontal.
O atributo rowspan mescla células na vertical.
Os atributos width e height definem a altura e a largura da tabela.
