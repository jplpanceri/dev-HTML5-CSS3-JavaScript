Imagens

Imagens são representadas pela tag img.

Possui os atributos:
src que é onde se coloca o link da imagem que pode ser de uma página web ou pode-se colocar o endereço de uma imagem que está dentro do computador.
alt que exibe um texto caso a imagem não possa ser exibida.
width largura e height altura da imagem podendo colocar um número qualquer ou porcentagem que vai ocupar da tela.

Links

A tag a é utilizada para links. Dentro dela está o atributo href que é utilizada para por o link da página ou o endereço do arquivo html que se quer abrir. Caso não preencha com nada, ao clicar nesse link ele vai abrir a mesma página novamente
